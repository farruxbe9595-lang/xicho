# Xicho Test — GitHub Pages web-versiya

Bu papka GitHub Pages uchun tayyorlangan oddiy web test dasturidir.

## Tarkib

- `index.html` — asosiy sahifa
- `style.css` — dizayn
- `script.js` — test logikasi
- `question_bank.js` — 22 ta yo‘nalish va 972 ta savol bazasi
- `data_summary.json` — qisqa statistika

## GitHub Pagesga joylash

1. GitHub’da `xicho` nomli yangi repository oching.
2. Ushbu ZIP ichidagi fayllarni repository’ning asosiy joyiga yuklang.
3. GitHub → Settings → Pages bo‘limiga kiring.
4. Source: `Deploy from a branch`.
5. Branch: `main`, Folder: `/root` tanlang va Save bosing.
6. Bir ozdan keyin link shunday ko‘rinishda ishlaydi:

```text
https://USERNAME.github.io/xicho/
```

## Savollarni yangilash

Savollar `question_bank.js` faylida turadi. Yangi Excel fayllardan bazani qayta chiqarib, shu faylni almashtirish kifoya.
